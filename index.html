Projet BTS - Application de gestion de frais

Ce projet est l’une de mes réalisations les plus représentatives dans le cadre du BTS SIO. J’y ai travaillé sur une application de gestion des frais avec une partie web et une partie mobile. Le projet comprend une API REST en PHP, un système d’authentification par jeton, une connexion à une base MySQL, ainsi qu’une communication avec une application Android développée en Java à l’aide de Volley. Il démontre ma capacité à relier front-end, back-end, données, sécurité et tests d’intégration.

API REST PHP Jeton MySQL Android Volley

Architecture

Sécurité

Tests d’intégration entre API et applications clientes pour assurer le bon fonctionnement global.
